# Advent of Code 2024

[Advent of Code](https://adventofcode.com/)

[Hatch Leaderborad](https://adventofcode.com/2024/leaderboard/private/view/793510)

## Requirements

- GNU/linux tools (e.g. `find`)
- Make
- GCC (`c2x`)
- Python 3
- xxd

```
sudo apt-get install -y xxd
```

## Build

```sh
# Build all
make all

# Build just 1-1
make 1-1

# Clean generated output
make clean
```

## Run

```sh
build/bin/1-1

build/bin/2-1

# etc...
```
